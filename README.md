### Usage

![](https://img.shields.io/npm/v/@jimengio/code-rule-editor.svg?style=flat-square)

```bash
yarn add @jimengio/code-rule-editor
```

```tsx
let [segments, setSegments] = useState([]);

<CodeRuleEditor
  segments={[]}
  onSubmit={(xs) => {
    setSegments(xs);
  }}
/>;
```
